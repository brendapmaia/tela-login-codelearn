# Tela de Login - Code Learn 💻

Este projeto é uma tela de login responsiva, com exibição dinâmica de senha (olhinho). Desenvolvido com HTML, CSS e JavaScript puro.

🔍 **Funcionalidades:**
- Campo de e-mail e senha
- Ícone de "olhinho" que alterna a visibilidade da senha
- Estilo moderno com responsividade
- Design simples e funcional

![Gif demonstrativo](coloque-o-link-do-gif-aqui.gif)

🔧 **Tecnologias utilizadas:**
- HTML5
- CSS3
- JavaScript
- Font Awesome
